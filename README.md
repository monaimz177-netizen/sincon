# 🔱 SIN-con v500.0 | THE APEX PREDATOR

**SIN-con** هو الجيل الخامس من محركات معالجة البيانات النصية الضخمة. صُمم هذا المحرك ليكون "الثقب الأسود" للبيانات؛ حيث يقوم بسحق الملفات النصية إلى أصغر حجم فيزيائي ممكن مع تغليفها بطبقة حماية تمنع التعرف عليها من قبل برامج الضغط التقليدية.



## 🚀 المميزات الخارقة (Apex Features)

* **Self-Adaptive Engine:** يكتشف السكربت عتاد جهازك (Android, Linux, Windows) ويضبط قوة المحرك تلقائياً ليناسب حجم الرام المتاح.
* **The .sin Vault:** صيغة ملفات فريدة ومشفرة. لا يمكن لبرامج مثل WinRAR أو 7-Zip فك ضغطها؛ هي ملك لك ولـ SIN-con فقط.
* **Atomic Squeezing:** استخدام خوارزميات LZMA المتقدمة مع فلاتر دلتا لضمان أعلى نسبة ضغط للملفات النصية و Wordlists.
* **Streaming Protection:** يعالج الملفات كتدفقات (Chunks)، مما يعني أنه يمكنه معالجة ملفات بحجم 100GB على أجهزة ضعيفة دون انهيار.
* **Secure Registry:** نظام سجلات مدمج بقاعدة بيانات SQLite لتوثيق جميع عملياتك بنجاح.

## 🛠️ المتطلبات (Requirements)

السكربت ذكي بما يكفي ليقوم بتثبيت متطلباته بنفسه، ولكن للتأكد من أفضل أداء:
* Python 3.x
* مكتبة `psutil` (سيحاول السكربت تثبيتها تلقائياً عند أول تشغيل).
